# knit_gauge — 編み物ゲージ方眼紙メーカー

編み物の製図を助けるWebアプリです。ゲージ(目数・段数)を入力すると、**1マスが実寸の1目×1段** になる方眼紙を作成できます。印刷してその上に直接製図できます。

🔗 **公開URL**: https://mihohoi0322.github.io/knit_gauge/

## 機能

- ゲージ入力: 「10cmあたり」(一般的なゲージ表記) と「1cmあたり」を切替可能
- リアルタイムプレビュー: 入力するとすぐに方眼が表示される
- **PDF出力(実寸)**: A4 1枚に収まる方眼をmm単位の精度で出力。印刷時に「実際のサイズ」を選べば、1マスが実際の1目・1段のサイズになります
- 用紙の向き(A4 縦/横)、太線の間隔(5マス/10マス/なし)を選択可能
- ゲージ情報は方眼の右上に表示されます

## 使い方

1. ゲージの目数・段数を入力する(例: 10cmあたり 20目 28段)
2. 用紙の向きと太線の間隔を選ぶ
3. 「PDFをダウンロード」を押す
4. PDFを印刷するときは、拡大縮小せず「実際のサイズ」で印刷する

## 開発

```bash
npm install
npm run dev    # 開発サーバー起動
npm run build  # 本番ビルド (dist/)
```

- Vanilla JS + [Vite](https://vitejs.dev/)
- PDF生成: [jsPDF](https://github.com/parallax/jsPDF)(mm単位で直接描画するため実寸精度が保たれます)
- `main` ブランチへの push で GitHub Actions が自動的に GitHub Pages へデプロイします
